# RaceMenu Selector of Skins — Baby Girl Edition
### 99 Skin Slots · CBBE 3BA · UBE · Face + Body + Hands

A heavily expanded version of [RaceMenu Selector of Skins](https://www.nexusmods.com/skyrimspecialedition/mods/126721), grown from 4 slots to **99 skin slots** — each one a different CBBE female skin texture set packed in automatically from the archives.

The MCM slider in RaceMenu handles the swap live, no save reload needed. Slot 0 reverts to your default character skin. Every slot swaps face, body, and hands textures simultaneously. Player-only — NPCs are completely unaffected. Selected slot persists through saves, loads, and cell transitions.

---

## Features

- 99 selectable skin slots via the **Body** category in RaceMenu
- Every slot swaps **face, body, and hands** textures simultaneously
- Slots 2–38 each pull textures from a specific skin mod — all pre-packed
- Slots 39–99 use CBBE base textures and are empty/ready for your own custom skins
- Slot 1 uses the **unique body mesh path** (same as Unique Character) for a fully custom body shape
- Body meshes included for **CBBE 3BA, UBE, and Unique** presets
- Papyrus script driven — no INI edits, no patches required
- Compatible with OBody, Wet Function Redux, and pregnancy mods

## Installation

> **This archive does NOT include texture DDS files.** You need to source them from the skin mods listed above, or use the included scripts to extract them automatically.

### Option A — Manual

1. Install the mod as normal via MO2 or Vortex
2. For each slot, place the skin's textures into `textures/aixbodyselector/skinXX/female/`
3. Each slot needs 13 DDS files:

```
femalehead.dds          femalehead_msn.dds      femalehead_s.dds        femalehead_sk.dds
femalebody_1.dds        femalebody_1_msn.dds    femalebody_1_s.dds      femalebody_1_sk.dds
femalehands_1.dds       femalehands_1_msn.dds   femalehands_1_s.dds     femalehands_1_sk.dds
blankdetailmap.dds
```

### Option B — Auto-install scripts (recommended)

1. Place your skin mod archives in a `skin/` subfolder inside your MO2 downloads folder
2. Edit the paths at the top of `scripts/install_body.py` and `scripts/install_skins2.py`
3. Run both scripts — they auto-detect and extract the correct textures, falling back to CBBE for any missing files

---

## Adding Your Own Skins (Slots 39–99)

Each empty slot already has a full ESP record chain (Armor → ArmorAddon → TextureSet) pointing to its folder. Just drop your 13 DDS files into the matching slot folder:

```
textures\aixbodyselector\skinXX\female\femalehead.dds
textures\aixbodyselector\skinXX\female\femalebody_1.dds
textures\aixbodyselector\skinXX\female\femalehands_1.dds
(+ _msn, _s, _sk variants for each, plus blankdetailmap.dds)
```

You can also replace textures while the game is running — the swap takes effect the next time you change slots in RaceMenu.

## Load Order & Compatibility

- No vanilla records edited — place anywhere in load order
- **MO2 priority:** This mod must sit below (higher priority than) any skin mod whose textures you want it to override
- **OBody NG** — Compatible. OBody handles body shape; this mod handles skin textures
- **Wet Function Redux** — Compatible within the same skin family; brief seams possible when switching between very different skins
- **Racial Skin Variance** — Use the version that does not touch the player character
- **Unique Character** — Redundant; this mod covers everything Unique Character does for humanoid races

## Scripts

| Script | Purpose |
|--------|---------|
| `install_skins2.py` | Extracts face textures (femalehead × 4) from all 37 skin archives |

| `fix_failed_slots.py` | Re-tries slots that fell back to CBBE due to path-with-spaces issues |
| `survey_body.py` | Surveys all archives to preview which textures are available |
